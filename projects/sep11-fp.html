Maze Game

For this project I decided to make a maze game using Threejs which is a JavaScript library that allows users to render 3D objects. It was a enjoyable library to use and it was for me since it allowed me to be creative in many ways and the idea of creating a maze was a great idea.

Challenges

One big challenge that I had during this project was the use of the documentation. The documentation wasn't the clearest and so that was one issue that I had. I struggled with the documentation and instead resorted to youtube tutorials and I had an easier time learning Threejs.

Takeaways

One major takeaway that I can apply for the future is to find ways to make things easier. Not like searching answers but learn ways to make your learning easier or make the overall process of making something easier. Similar how I used youtube tutorials instead of the documentation, this made the learning process much easier and got me my product.
